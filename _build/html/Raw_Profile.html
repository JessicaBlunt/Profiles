Raw_Profile¶

Reads data file (JSON or netCDF) and stores the raw data

Authors Brian Greene, Jessica Wiedemeier, Tyler Bell, Gus Azevedo

Copyright University of Oklahoma Center for Autonomous Sensing and Sampling 2019

class Raw_Profile.Raw_Profile(file_path, dev=False)¶

Contains data from one file.

Variables:
  • temp (tuple) – temperature as (voltage1, voltage2, …, time: ms)
  • rh (tuple) – relative humidity as (rh1, rh2, …, time: ms)
  • temp_rh (tuple) – temperature as (voltage1, voltage2, …, time: ms)
  • co2 (tuple) – CO2 data as (CO2, CO2, …, time: ms)
  • gps (tuple) – GPS data as (lat, lon, alt_MSL, time: Datetime)
  • pres (tuple) – barometer data as (pres, time: ms)
  • rotation (tuple) – UAS position data as (roll, pitch, yaw, time:ms)
  • dev (bool) – True if the data is from a developmental flight
co2_data()¶

Gets data needed by the CO2_Profile constructor.

rtype: list return: [co2, temp, rh]

read_netCDF()¶

Reads data from a NetCDF file. Called by the constructor.

thermo_data()¶

Gets data needed by the Thermo_Profile constructor.

rtype: list return: [temp, rh, pres]

wind_data()¶

Gets data needed by the Wind_Profile constructor.

rtype: list return: [rotation]
