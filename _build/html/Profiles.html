Profiles (COMING SOON)¶

Manages data from a collection of flights or profiles at a specific location

Authors Brian Greene, Jessica Wiedemeier, Tyler Bell, Gus Azevedo

Copyright University of Oklahoma Center for Autonomous Sensing and Sampling 2019

class Profiles.Profiles(descent=False, dev=False)¶

This class manages data (in the form of Profile objects) from one or many flights.

Variables:
  • location (Location) – information about the location of the flights
  • profiles (list<Profile>) – list of Profile objects at this location
  • descent (bool) – True if a) data is to be trimmed to single vertical profiles, and b) data from the descending (rather than ascending) leg is to be used.
  • dev (bool) – True if data from developmental flights is to be uploaded
