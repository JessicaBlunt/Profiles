plotting (COMING SOON)¶

fpath_logos¶

The file path to the logos used in plotting.summary. Alterations are not recommended if the standard directory structure is used.

plotting.contour_height_time(profiles, var1='temp', var2=None)¶

contourHeightTime creates a filled contour plot of var1 in a time-height coordinate system. If var2 is not None, it also overlays unfilled contours of var2.

Parameters:
  • profiles (list) – a list of all profiles to be included in the plot
  • var1 (Var) – the name of the variable to be filled
  • var2 (Var) – the name of the variable to be drawn
Raises:

TypeError – if var1 is not a valid instance of Var
Return type:

matplotlib.figure.Figure
Returns:

the contoured plot
plotting.meteogram(fpath)¶

Graphically displays Mesonet data.

Four subplots are created, each with time on the horizontal axis. The top plot is of T and Td, the second of P, the third of wind speed and direction, and the bottom of solar radiation.

Parameters:fpath (string) – the file path for the Mesonet timeseries data
Return type:matplotlib.figure.Figure
Returns:figure containing four horizontal subplots

Keyword Arguments:

plotting.summary(temp=None, pres=None, t_d=None, u=None, v=None, dt=None, loc=(None, None), flight_name=None)¶

Creates a figure with a SkewT, hodograph, map, and logos. The plot title will be <flight_name> at <loc>, <time>. Logos can be changed by altering the contents of directory resources.

Parameters:
  • temp (list<number>) – Temperatures to plot in C
  • pres (list<number>) – Pressures to plot in ?
  • t_d (list<number>) – Dewpoints to plot in C
  • u (list<number>) – U-component of wind in kts
  • v (list<number>) – V-component of wind in kts
  • dt (datetime) – start time of flight (for title)
  • loc (tuple<number>) – lat, lon pair of location data
  • flight_name (string) – name of the flight (for title)
Return type:

matplotlib.figure.Figure
Returns:

the summary figure (described above)
